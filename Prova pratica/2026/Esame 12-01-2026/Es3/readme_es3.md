# README - Esercizio 3 (no8cat)

Questo script Python esegue la copia di un file di testo con codifica UTF-8. Durante la copia, tutti i caratteri che non appartengono allo standard ASCII vengono sostituiti con il simbolo `?`.

## Utilizzo

Lo script è progettato per essere eseguito da riga di comando e accetta esattamente due argomenti posizionali:
1. Il percorso del file di input da leggere.
2. Il percorso del file di output da generare.

**Sintassi:**
```bash
python3 no8cat.py <filein> <fileout>
